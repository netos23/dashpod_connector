# Dashpod
___

Dashpod is delivery system for managing plugins and updates for your app instantly 

## Structure

* [dashpod](.) - main package
* [dashpod_api](../dashpod_api) - API for Dashpod
* [dashpod_delivery](../dashpod_delivery) - Plugins delivery system
* [dashpod_updater](../dashpod_updater) - App updater components
